Project 2

Feasibility Analysis of Server Virtualization

Financial Analysis

Project Overview

In this project, I acted as a System Administrator for a fictitious organization, HostPenny, to conduct a comprehensive financial cost-benefit analysis for a large-scale server virtualization initiative.

  • Current State Assessment: I analyzed the operational costs of 20,000 physical servers divided into four functional groups, calculating energy consumption in kWh based on average CPU loads and power rates.
  • Consolidation Strategy: I proposed a 10.2:1 physical consolidation ratio, reducing the hardware footprint to 1,960 high-performance Sun X9-2L servers capable of supporting 3,830 virtual machines.
  • Financial Projections: I developed a 5-year financial projection including hard costs (server purchase, maintenance, and software licenses) and soft costs (administrative effort and backup labor).
  • Dynamic Modeling: I constructed a series of interconnected spreadsheets using shared variables to ensure the feasibility study remained flexible and free of hard-coded numerical values.
