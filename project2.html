Project 2

Server Virtualization & Administration

System Administration

Project Overview

In this project, I acted as a System Administrator for a fictitious organization, HostPenny, to conduct a comprehensive financial cost-benefit analysis for a large-scale server virtualization initiative.

  • CVirtual Private Network Deployment: I architected and configured a secured virtual private network SAVN.local using Oracle VirtualBox, implementing static IPv4 addressing and internal networking to facilitate communication.
  • Financial Projections: I developed a 5-year financial projection including hard costs (server purchase, maintenance, and software licenses) and soft costs (administrative effort and backup labor).
  • Active Directory Management: Set up a Domain Controller with AD DS and DNS roles to manage centralized security and user authentication through Organizational Units.
