Project 1: Three-Tier Distributed Web-Based Application


For this project, I developed a sophisticated three-tier distributed web application designed to manage a complex suppliers/parts/jobs/shipments database[cite: 2, 4, 5]. The application utilizes a modular architecture to separate the user interface, server-side logic, and persistent data storage.

Key technical implementations included:
  • First-Tier (Front-End): Developed dynamic JSP and HTML landing pages to provide user authentication and distinct interfaces for Root, Client, and Accountant-level users[cite: 14, 21, 24, 25].

  • Second-Tier (Application Logic): Utilized Java Servlets running on a Tomcat container to process arbitrary SQL commands and implement complex server-side business logic[cite: 4, 37, 40]. This tier handles automatic supplier status increments based on shipment quantities without relying on database triggers[cite: 39, 41, 65].

  • Third-Tier (Back-End): Managed a persistent MySQL database with enforced referential integrity and foreign key constraints[cite: 10, 49]. I also implemented remote stored procedures accessed via the CallableStatement interface for specialized accountant reports[cite: 30, 47, 48].

  • Secure Connectivity: Managed database connections through properties files to ensure secure, credential-based access for different user privilege levels[cite: 32, 33, 82].
