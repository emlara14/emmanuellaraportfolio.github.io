Project 1

Three-Tier Distributed Web-Based Application

Project Overview

I developed a distributed three-tier web application using Java servlets and JSP technology running on a Tomcat container to maintain a persistent MySQL database.

  • Multi-Level Authentication: An HTML landing page facilitates user authentication via a servlet that validates credentials against a dedicated MySQL credentialsDB. Users are redirected to specific interfaces for root, client, or accountant roles.
  • Logic & SQL Interface: Second-tier servlets handle arbitrary SQL processing for root and client users. I implemented server-side business logic to automatically increment supplier status by 5 when shipment quantities reach 100 or more, specifically without using database triggers.
  • Data & Reports: The backend MySQL DBMS enforces referential integrity for shipment records. Accountant-level reports are generated by executing remote stored procedures via the CallableStatement interface.
  • Secure Access: Database connections are managed through properties files to ensure distinct credential-based access for root, client, and accountant privilege levels.
